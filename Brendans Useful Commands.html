Useful Vim Commands

Inline Commands What it does
h j k l Basic movement keys.
i a I A Enter insert mode (insert at cursor/append after cursor/insert at beginning of line/append to end of line).
Visual Mode Commands What it does
Ctrl-v block-wise visual mode
Shift+V line-wise visual mode
Ctrl+V + select lines + Shift+I + [make edits] + esc Columnar select
Search Commands What it does
/[keyword] Searches for keyword in the document
n N Searches your text again in the same/opposite direction of your last search
:%s/foo/bar/g Find each occurrence of 'foo' (in all lines), and replace it with 'bar'.
